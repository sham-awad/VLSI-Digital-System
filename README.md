# VLSI Digital System

## Overview

This project was developed as part of the Digital Electronics Lab course. It implements a CMOS digital system that integrates a synchronous counter, a 6T SRAM memory array, D flip-flops, and a half adder.

The design was implemented using Electric VLSI Design System for layout design and LTspice for simulation and waveform verification.

## Features

- 2-bit Synchronous Counter
- 6T SRAM Memory Array
- D Flip-Flops
- Half Adder
- Top-Level CMOS Layout
- LTspice Simulation
- Layout Verification

## Tools Used

- Electric VLSI Design System
- LTspice
